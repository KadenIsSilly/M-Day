# Advanced Warfare: Exo Zombies - No Mutated Zombies Mod

**Features**
- Removes the respective mutated zombie.
- Doesn't mess with any regular zombie spawns.
- Compatible with all maps.

**Installation**
1. Download [Alterware](https://alterware.dev/).
2. Download the scripts from [releases](https://github.com/KadenIsSilly/M-Day/releases).
3. Extract all the files into `\data\scripts\mp\zombies`
